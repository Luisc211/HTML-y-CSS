Flex Wrap

Flex Wrap es una propiedad de CSS que usamos dentro de una caja contenedera con display: flex, para indicar si las cajas hijas flexibles deben mantenerse en una sola línea o pueden pasar a varias líneas cuando ya no caben en el espacio disponible.

Cajas sin flex wrap, valor por defecto

Si la cantidad de cajas hijas exceden el ancho de la caja padre, las que no caben se desbordaran y se saldran del contenedor
