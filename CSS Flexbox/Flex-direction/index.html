Flexbox

Flexbox es un modelo de caja unidimensional en CSS, que nos ayuda a organizar elementos ya sea en fila o en columna, alinearlos facilmente, y controlar el espacio entre ellos sin usar margenes o posiciones complicadas, para que flexbox exista debe haber una caja contenedora que es a la que se le activara display flex, y las cajas contenidas, o flex items, algunas propiedades se le deben dar a la caja padre, otra a las cajas hijas, pero estas cajas deben ser hijas directas.
