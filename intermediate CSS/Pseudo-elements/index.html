Pseudoelementos en CSS

  • ::Before y afterInsertan contenido antes o después del elemento
  • ::First-lineEstiliza la primera linea de textos
  • ::First-letterEstiliza la primera letra (Ideal para capitulares)
  • ::SelectionEstiliza texto seleccionado por el usuario
  • input::placeholderEstiliza los placeholder de los inputs
  • ::markerEstiliza los marcadores de listas: Bullets, viñetas y números
  • ::backdropEstiliza el fondo cuando un elemento está en fullscreen o en un dialog:
