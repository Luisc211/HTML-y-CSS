La etiqueta h1 representa el título principal de la página y solo debe usarse una vez por página

La etiqueta h2 representa los títulos de secciones principales

La etiqueta h3 representa los subtítulos dentro de esas secciones

La etiqueta h4 representa subtítulos más detallados dentro de un h3

La etiqueta h5 representa detalles muy específicos dentro de subsecciones
La etiqueta h6 representa notas o subdetalles muy específicos

Este es un párrafo de ejemplo. Sirve para escribir contenido normal en la página web.
