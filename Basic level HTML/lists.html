Página de listas

En HTML tenemos dos tipos de listas: ol (ordered list), o en español listas ordenadas, y ul (unordered list), o en español listas desordenadas. Ambas etiquetas contienen elementos hijos llamados li (list item), que envuelven cada elemento que queremos listar. Si el orden es importante, usamos ol. Si el orden no importa, usamos ul.

Ejemplo de una lista ordenada con la etiqueta ol

  1. Elemento 01
  2. Elemento 02
  3. Elemento 03
  4. Elemento 04

Ejemplo de una lista desordenada con la etiqueta ul

  • Elemento A
  • Elemento B
  • Elemento C
  • Elemento D
